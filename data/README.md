# Data Access Instructions

Due to licensing and size constraints, the raw clinical data is not hosted directly in this repository. To reproduce the validation results, please obtain the source data:

1. **Source:** TCGA Pan-Cancer Atlas (Stomach Adenocarcinoma)
2. **Portal:** [cBioPortal](https://www.cbioportal.org/)
3. **Instructions:**
   * Select "Stomach Adenocarcinoma (TCGA, PanCanAtlas)"
   * Click "Download"
   * Select "Clinical Data"
   * Rename the clinical data file to `tcga_2018_clinical_data.tsv` and place it in this directory.

*Note: A sample file is included in this directory for demonstration purposes.*
